# Model-free pressure sensor placement in wáter distribution networks

Implementation of the method used in the paper

> L. Romero-Ben, G. Cembrano, V. Puig, and J. Blesa,
“Model-free Sensor Placement for Water Distribution Networks using Genetic Algorithms and Clustering,”
IFAC-PapersOnLine, vol. 55, n. 33, pp. 54--59, 2022.
